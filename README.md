## Tips

### Shapeshift Tips
- Don't rush at the beginning - focus on accuracy first
- Once you get the hang of it, try to improve your response time
- Pay attention to your accuracy percentage
- The colorful shapes make it more engaging but don't let them distract you

### Code-Compare Tips
- **Use the audio cues**: Listen to the unique sounds for each digit (0-9)
- **Create mental chunks**: Break the 7-character code into smaller segments
- **Visual scanning**: Quickly scan for obvious differences between options
- **Stay calm under pressure**: Time limits get shorter, but accuracy is key
- **Practice pattern recognition**: Look for common letter/number sequences

### General Tips
- Use the stop button if you need a break
- Focus on consistency over speed initially
- Track your improvement over multiple sessions
